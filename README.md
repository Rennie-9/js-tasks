# Решение задач по JavaScript

Мой личный проект для прокачки навыков программирования. Прохожу задачник по уровням, пишу чистый код и изучаю профессиональные инструменты разработки.

## 🚀 Мой прогресс по уровням:

### Уровень 1: Основы языка
- [x] [Уровень 1.1: Базовые условия и строки](./Level_1/Level_1_1/tasks.js) — *Реализованы проверки чисел, чётность, вычисление длины строк и сравнение первых символов.*
- [x] [Уровень 1.2: Основы работы с цифрами](./Level_1/Level_1_2/tasks.js) — *Реализовано извлечение первой/последней цифры числа через индексы строк, математическое сложение динамических данных и расчет длины.*
- [x] [Уровень 1.3: Работа с условиями и остатком от деления](./Level_1/Level_1_3/tasks.js) — *Реализованы проверки деления без остатка, анализ четности, валидация пользовательского ввода и ветвление логики.*
- [x] [Уровень 1.4: Работа с циклами](./Level_1/Level_1_4/tasks.js) — *Реализован вывод числовых диапазонов в прямом и обратном порядке, а также фильтрация чисел по четности и кратности с помощью цикла for.*
- [x] [Уровень 1.5: Накопление значений в циклах](./Level_1/Level_1_5/tasks.js) — *Реализовано суммирование числовых диапазонов с фильтрацией по четности/нечетности, расчет остатка от деления и посимвольный реверс строк.*
- [x] [Уровень 1.6: Работа с массивами и циклом for...of](./Level_1/Level_1_6/tasks.js) — *Реализован динамический сбор массива данных через prompt с условием остановки, расчет суммы квадратов, квадратных корней и фильтрация элементов по диапазонам.*
- [x] [Уровень 1.7: Фильтрация строк и реверс данных](./Level_1/Level_1_7/tasks.js) — *Реализовано разделение строки на массивы букв и цифр, математический реверс последовательности и суммирование числовых элементов.*
- [x] [Уровень 1.8: Интерактивный конвейер обработки чисел](./Level_1/Level_1_8/tasks.js) — *Создано консольное приложение (CLI) с бесконечным циклом `while` и динамическим распределением данных по трем массивам «на лету» (валидация диапазонов, четности и кастомное побитовое округление дробей).*
